Linear Models

Posted by Madhavi Jouhari on So. 25 Juli 2021

Chances are if you to read explanations of what a Linear Model does in Data Science paradigm, a lot of heavy mathematical terms will be thrown at you. There is an equal probability that you may or may not understand it, so here is more watered-down, Explain-Like-I'm-5 summarization to demystify these concepts.

Let me Fit those Curves for you

You know that a model learns the logic behind the dataset in the training process. Behind the curtains, the model is "fitting" the training data or trying to learn its origins using a function y(). 

y(w,x) = w_0 +w_1 x + ... + w_M x^M

In the above equation, the function is of polynomial nature.

  1. Here, M is the order of this polynomial function.
  2. The coefficients w_i is denoted by vector w

Although this polynomial is a non-linear function of x, it is a LINEAR function of the coefficients w.

Such functions (eg Polynomial), which are linear in unknown (coeff) parameters, have important properties and are called ...

(drumrolls)

... LINEAR MODELS

The value of w (coeffs) is learnt by fitting the function to the training data.

Now the secret behind the magical process of learning is done by minimizing an Error Function.

So umm 'What does this Error function do', you ask?

It measures the misfits between the function y(x,w), for any given value w, and the training set datapoints.

Error Function is a NON-NEGATIVE quantity : which means it can be 0 IF AND ONLY IF the function y(x,w) plot would pass exactly though each training data point in the graph.

So the curve resulting from plotting the y(x,w) function should fit into the pattern of the training data spread in the plane.

And thats what you call a CURVE FITTING bruh.

A very common Error Function is Sum of the Squares of the Errors between the predictions y(x_n,w) for each training datapoint x_n and its corresponding target value t_n. In this case, we minimize

E(w) = 1/2 Sigma n->N {y(x_n,w) - t_n}^2

The learning happens when this curve fits properly by choosing value of w for which E(w) is as close to 0 as possible.

Since the E(w) is a quadratic function of w -> Its derivative w.r.t. w will be linear in elements of w.

That means the process of MINIMIZATION OF ERROR function will yield a unique solution.
