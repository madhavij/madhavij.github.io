Update a JSON file with Python

I thought opening a JSON file was tricky. Understanding loads and dumps and the erross that came along were felt like head scratchers.

Then I had to make changes in the json file in an automated manner. It took me quite some time, in the first attempt, so here I am documenting the logic.

Lets say this is my simple json stored in a text file - data.json .

{
  "id": "000",
  "Name": "Harvey Specter",
  "Age": "32",
}

And here I am loading the file into a python structure, editing it, iterating to the beginning of file and dumping back the new content.

import json

with open('data.json','r+') as filecontent:
    json_data = json.load(filecontent)
    json_data['id'] = '123'
    f.seek(0)
    f.write(json.dumps(json_data))
    f.truncate()

This is the simplest example I could think of. Hope it works out for you as well.
